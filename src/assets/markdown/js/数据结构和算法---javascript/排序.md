# 排序
#### 冒泡排序
```
function bubbleSort(array){
  var length = array.length;
  for (var i=0; i<length; i++){
        for (var j=0; j<length-1; j++ ){ 
            if (array[j] > array[j+1]){
                [array[j], array[j+1]] = [array[j+1], array[j]]
            }
        }
    }
    return array;
};
```
